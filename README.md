# Development env setup

- Setup Docker

- `git clone https://github.com/jirkavrba/studentsky-portal-fis-vse`

- `cd studentsky-portal-fis-vse`

- `docker build -t portal .`

- `docker-compose build`

- `docker-compose up`

- Run migrations
